# Handwritten Character Recognition

## Objective
Identify handwritten characters or alphabets using image processing and deep learning.

## Approach
Image processing combined with a Convolutional Neural Network (CNN) to classify handwritten digits and characters.

## Key Features
- Dataset: MNIST (digits), EMNIST (characters)
- Model: Convolutional Neural Network (CNN)
- Extendable to full word or sentence recognition using sequence modeling (e.g. CRNN)

## Evaluation Metrics
- Accuracy
- Loss
- Confusion Matrix

## Tools Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
